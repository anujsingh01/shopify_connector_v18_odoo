MODULE REQUIREMENTS

The module requires the Shopify API python library named   ShopifyAPI.


Installation instructions

  1. Place all downloaded modules into your Odoo's addons folder.
  2. Install required python modules: pip install ShopifyAPI as odoo user.
  3. Go to "Apps" (in debug mode) and click "Update Apps List". Accept at confirmation.
  4. Search for "Odoo Shopify Connector" in Apps list and install it by pressing "Activate" button.

Features

Setup & Import

  • Comprehensive initial data import
  • Import product details including prices, categories, attributes, and more.
  • Import existing customer information, including addresses and contact details.
  • Import historical orders data with all relevant details.
  • Apply inventory levels from your Shopify store to Odoo.

Automated products mapping

  • Automatically map products between Shopify and Odoo using SKU/Internal Reference fields.
  • Validation test helps to fix any issues with the product catalog to avoid problems during the use of the Shopify connector.

Import and Export Products

  • Import product details including categories, attributes, and more from Shopify to Odoo.
  • Export new products from Odoo to Shopify with ease.
  • Customize fields that need to be synchronized (for export only, import only, or in both directions).
  • Synchronize custom fields (with some limitations).
  • Synchronize product images in both directions (can be disabled if needed).
  • Exclude specific products from synchronization based on your needs.
  • Support for product webhooks to handle real-time updates from Shopify to Odoo.

Automatic Updates

  • Automatically export products when they are changed in Odoo (e.g., when the price is changed).
  • Ability to set Odoo pricelist to be used during product prices export.

Inventory Control

Flexible Inventory Synchronization:

  • Automatically synchronize stock levels between Shopify and Odoo in real-time or periodically, providing flexibility based on your business needs.

Configurable Synchronization Options

  • Define which quantity field should be synchronized: Free to Use Quantity, On Hand Quantity, or Forecasted Quantity, based on your business needs.
  • Option to exclude specific products from stock synchronization, giving you more control over your inventory management.

Multi-Location Support

  • Synchronize product quantities from different Odoo locations to corresponding Shopify locations, allowing for accurate inventory management across multiple warehouses.

Order Processing

Order Import

  • Import orders from Shopify to Odoo with all relevant details, including customer information, products, payment and shipping methods.
  • Apply discounts from Shopify orders to Odoo orders accurately, including delivery line discounts.
  • Automatically create products, delivery carriers, and taxes in Odoo if they do not exist yet.
  • Import fraud scores from Shopify and mark orders as risky if the fraud score exceeds a specified threshold.

Order Management

  • Support for order webhooks to handle real-time updates from Shopify to Odoo.
  • Automatically update order status in Odoo based on changes in Shopify, such as order cancellations or fulfillment status.

Auto-Workflow for Orders

  • Automatically trigger actions such as orders confirmation, invoicing, shipping, etc. based on predefined conditions.
  • Customize workflow steps to match your business processes, reducing manual intervention and increasing efficiency.

Customer Data

Customer Import

  • Import customer information, including names, addresses, and contact details from Shopify to Odoo.
  • Synchronize Shopify customer metafields ensuring all relevant customer information is included.
  • Maintain a structured and clean customer catalog by handling contacts in an "Odoo way". This ensures alignment with Odoo workflows for orders and transfers.

Customer Mapping

  • Connector uses mappings to avoid the creation of duplicated contacts in Odoo. When mapping is missing (new customer), the connector will try to find existing contacts using all available information (email, name, phone, etc.) before creating a new Odoo contact.
  • Manually map customers between Shopify and Odoo for businesses working with a small set of customers, providing additional control over your customer data.
